A festive Python Turtle scene! 
It draws a Christmas tree with ornaments, garlands, gifts and a sleigh.

## Features

- Symmetric, layered **Christmas tree** with a star on top.
- Colorful **ornaments** and wavy golden **garlands**.
- Wrapped **gifts** under the tree.
- Red **sleigh** with runners and wheels.

## Tech Stack

- Python 3.13
- `turtle` (standard library)
- `random`, `math`, `argparse`, `os`
- `playsound` for audio (optional)

clone the repo
